How to Use This Site

⚡ Quick Start

1. Pick events (left sidebar) 2. Set reminders (bell) 3. Mark done (✓) 4. Optional: add API key for auto‑mark.

🧭 Icons

✓ done API badge = auto‑marked ⭐ favorite 🔔 reminder 🕒 weekly/special auto‑alert ✔ checklist auto‑mark 🔗 chain mode 📖 wiki [fast] guide 📍 waypoint.

🔔 Alerts & Chains

Card bell: 2m → 5m → 10m → off. Clock: keeps a rolling 5m reminder on the next incomplete weekly/special WV meta. Chain mode: marking one chain step auto‑adds a short reminder to the next steps.

🎮 API & Auto‑Mark

Add an API key (account + progression) to auto‑mark WV world boss / meta objectives (daily, weekly, special). Checklist toggle enables/disables marking; weekly clock toggle is independent. Rescan (refresh icon) forces a new WV + achievement sync.

⭐ Favorites & Filters

Star marks all occurrences of that event. Header star = show only favorites. Header bell = show only events with an active reminder. Search filters by name or map instantly. Sidebar category toggles hide/show whole groups (use select all / none for speed).

🕐 Timing

You see start time plus a countdown (countdown appears in last 15 min). Events appear up to 5h early so you can plan ahead. Marked completions reset at daily reset (00:00 UTC); weeklies follow Monday reset.

🎨 Appearance

Generated backgrounds are lightweight procedural textures keyed per category. Switch to Plain in Appearance if you want neutral cards; regenerate to refresh cached images.

💡 Tips

Use chain mode for meta trains. Keep the weekly clock on for passive progress. Clear a reminder by cycling past 10m. API badge = verified objective progress (full or claimed). Favorites + search = fast personal shortlist.

🔐 Storage

Everything (key, filters, favorites, done flags, backgrounds) lives locally in your browser storage—nothing is uploaded here.
