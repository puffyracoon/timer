How to Use This Site

🕐 Timer Features

What are the different time displays?
Each event shows the start time (when it begins) and remaining time (countdown). Events appear 5 hours before they start.
Why do some events not show a countdown yet?
To reduce visual noise the countdown only appears in the final 15 minutes before an event starts. Before that you just see the start time.
How do I mark events as completed?
Click the checkmark ✓ icon to mark an event as done. This saves until the next daily reset and dims the event card.

🔔 Alerts & Notifications

How do I set up event alerts?
Click the bell 🔔 icon to cycle through alert times: 2m → 5m → 10m → off. You'll get browser notifications when enabled.
What are Event Chains?
The 🔗 button enables auto-alerts for meta event chains (like HoT train). When you complete one event, it automatically sets reminders for the next events in the sequence. Events that start a meta chain are marked with a 🔗 indicator next to their name.
Do chain alerts auto-set a reminder?
Yes. When chain alerts are enabled and you mark a chain event as done, the next events in that chain will auto-get a short (2m) reminder if they don't already have one. You can still click the bell to adjust or remove it.

🎯 Organization

How do I use favorites?
Click the star ⭐ to favorite events. Use the star button in the header to show only favorited events. Favoriting one instance marks all instances of that event.
How do I view my active alerts?
Click the bell 🔔 button in the header to show only events with active reminders set. This helps you see all your scheduled alerts at a glance.
How does the search work?
Type in the search box to filter events by name or map. Search works in real-time as you type.
How do I hide events I don't want?
Open the sidebar (hamburger menu) and use the toggles to hide entire event categories or individual events. Use "select all" / "select none" for quick category management.

🎮 Guild Wars 2 Integration

What is the GW2 API key for?
Enter your GW2 API key to enable:
  • World boss/meta auto‑marking (from your personal Wizard's Vault daily, weekly & special objectives)
  • Achievement-based completion for supported events
Only account + progression scopes are required. Auto‑marked events show an API badge and reset at 00:00 UTC (weekly/special objective availability may extend beyond a day).
Why aren’t WV boss/meta objectives marked without a key?
Boss/meta detection now scans /account/wizardsvault/daily, /weekly and /special, which all require your API key. Without a key we can’t see your personal objectives, so auto‑marking is disabled (you’ll see a banner prompting you to connect).
What does the Last Sync time mean?
It shows the most recent successful achievement fetch (UTC). Use the 🔄 Refresh button to manually sync; otherwise it auto-refreshes every 10 minutes.
What does "Auto-mark WV world boss/meta objectives" do?
When enabled (and API key present) the timer reads your personal Wizard's Vault daily, weekly & special objectives, matches any world boss or meta tasks via name variants, and auto‑marks those you’ve completed with an API badge. Disable anytime if you prefer manual tracking.
How do I force a new daily scan?
Click the 🔄 button in the header to re-fetch today's daily list and rebuild the boss mapping (clears the cached daily metadata for the day).
How do I copy waypoints?
Click the waypoint 📍 icon to copy the waypoint code. Paste it in GW2 chat to create a map marker.

🎨 Appearance

Why do the backgrounds look different?
Zone backgrounds are procedurally generated (abstract shapes + palette) so they are original and lightweight. Each category palette matches its expansion. They are cached locally for fast reloads.
Can I disable the generated backgrounds?
Yes. In the sidebar under Appearance choose Plain to switch to a simple neutral gradient. Switch back to Generated any time.
How do I regenerate a background?
Clear the related bg-* entries from your browser storage (or clear all site data) and reload, or temporarily switch to Plain and back after clearing to force a refresh.
