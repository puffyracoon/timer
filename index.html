How to Use This Site

🕐 Timer Features

What are the different time displays?
Each event shows the start time (when it begins) and remaining time (countdown). Events appear 5 hours before they start.
How do I mark events as completed?
Click the checkmark ✓ icon to mark an event as done. This saves until the next daily reset and dims the event card.

🔔 Alerts & Notifications

How do I set up event alerts?
Click the bell 🔔 icon to cycle through alert times: 2m → 5m → 10m → off. You'll get browser notifications when enabled.
What are Event Chains?
The 🔗 button enables auto-alerts for meta event chains (like HoT train). When you complete one event, it automatically sets reminders for the next events in the sequence. Events that start a meta chain are marked with a 🔗 indicator.

🎯 Organization

How do I use favorites?
Click the star ⭐ to favorite events. Use the star button in the header to show only favorited events. Favoriting one instance marks all instances of that event.
How do I view my active alerts?
Click the bell 🔔 button in the header to show only events with active reminders set. This helps you see all your scheduled alerts at a glance.
How does the search work?
Type in the search box to filter events by name or map. Search works in real-time as you type.
How do I hide events I don't want?
Open the sidebar (hamburger menu) and use the toggles to hide entire event categories or individual events. Use "select all" / "select none" for quick category management.

🎮 Guild Wars 2 Integration

What is the GW2 API key for?
Enter your GW2 API key in the sidebar to automatically mark events as completed based on your in-game achievements. No permissions needed - it only reads your achievement progress.
How do I copy waypoints?
Click the waypoint 📍 icon to copy the waypoint code. Paste it in GW2 chat to create a map marker.

🔗 Useful Links

What are the other icons for?
  • Wiki 📖 - Opens the event guide on Guild Wars 2 Wiki
  • [fast] 🚀 - Links to fast.farming-community.eu for detailed guides

💡 Pro Tips

  • Events are color-coded by expansion - green borders for HoT, orange for PoF, etc.
  • Events with a 🔗 indicator start meta farming chains - perfect for efficient farming
  • Icon hover effects match the event category colors
  • Use chain alerts for efficient meta farming
  • Use "select all/none" for quick category filtering
  • Your settings (timers, favorites, categories) are saved automatically in your browser
